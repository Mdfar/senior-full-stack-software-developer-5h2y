Gateway AI B2B Platform

Production-grade full-stack architecture for AI-powered onboarding.

Tech Stack

Frontend: Next.js 14, Tailwind CSS, TypeScript

Backend (API): Next.js API Routes (Edge) & FastAPI (Python)

Database: PostgreSQL with pgvector

AI Orchestration: LangChain & GPT-4o

Cloud: Google Cloud Platform (Cloud Run, GCS)

Getting Started

Install dependencies: npm install

Configure .env with cloud credentials.

Run development server: npm run dev
